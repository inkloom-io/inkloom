# Welcome to Acme

Acme SDK helps you build powerful documentation sites with ease.

{% hint style="info" %}
This guide assumes you have basic knowledge of JavaScript and Node.js.
{% endhint %}

{% hint style="warning" %}
Acme SDK requires Node.js 20 or later.
{% endhint %}

{% hint style="success" %}
You're in the right place to learn about Acme!
{% endhint %}

{% hint style="danger" %}

{% endhint %}

## Features

- Real-time collaboration
- Markdown and MDX support
- Built-in search
