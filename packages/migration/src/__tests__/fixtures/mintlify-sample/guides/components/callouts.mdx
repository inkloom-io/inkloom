## Using Callouts Effectively

1. Use **Note** for general supplementary information
2. Use **Warning** for potential issues or breaking changes
3. Use **Tip** for best practices and optimization hints
4. Use **Info** for contextual details
5. Use **Check** for success states and confirmations
