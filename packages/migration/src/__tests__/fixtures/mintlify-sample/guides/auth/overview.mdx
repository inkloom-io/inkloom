# Authentication Overview

Widget Platform supports multiple authentication methods.

<Info>
All API requests must include authentication credentials.
</Info>

## Choosing an Auth Method

<Tabs>
  <Tab title="Server-Side">
    Use JWT tokens for server-to-server communication. These tokens are long-lived and scoped to specific permissions.

    ```javascript
    const token = await client.auth.createToken({
      scope: ["read:projects", "write:data"],
    });
    ```
  </Tab>
  <Tab title="Client-Side">
    Use OAuth 2.0 for browser-based applications. This provides secure user authentication.

    ```javascript
    const auth = client.auth.oauth({
      clientId: "your-client-id",
      redirectUri: "https://your-app.com/callback",
    });
    ```
  </Tab>
</Tabs>
