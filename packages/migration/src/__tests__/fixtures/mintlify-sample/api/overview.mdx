# API Reference

The Widget REST API provides programmatic access to all platform features.

<Note>
All API requests require authentication. See the [Auth Overview](/guides/auth/overview).
</Note>

## Base URL

```
https://api.widget.io/v1
```

## Rate Limits

| Tier | Requests/min | Burst |
|------|-------------|-------|
| Free | 60 | 10 |
| Pro | 600 | 100 |
| Enterprise | 6000 | 1000 |

<Warning>
Exceeding rate limits returns a `429 Too Many Requests` response.
</Warning>
