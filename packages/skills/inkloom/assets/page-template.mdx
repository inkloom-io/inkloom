# Authentication

All API requests require authentication via an API key. You can generate keys from the dashboard under **Settings → API Keys**.

<Callout type="info">

API keys are scoped to your organization. Keep them secure and never commit them to version control.

</Callout>

Include your API key in the `Authorization` header of every request:

```bash title="Example request"
curl -X GET https://api.example.com/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY"
```
